requireEdition

source/index.js

Cycle through the editions and require the correct one

requireEdition(edition: edition, opts: Object): Any
Parameters
edition (edition) the edition entry
opts (Object) the following options
Name Description
opts.require string the require method of the calling module, used to ensure require paths remain correct
opts.cwd [string] if provided, this will be the cwd for entries
opts.entry [string] if provided, should be a relative or absolute path to the entry point of the edition
opts.name [string] if provided, should be the name of the package that we are loading the editions for
Returns
Any:
