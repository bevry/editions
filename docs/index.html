requirePackage

source/index.js

Cycle through the editions for a package and require the correct one

requirePackage(cwd: string, _require: function, customEntry: [string]): void
Parameters
cwd (string) the path of the package, used to load package.json:editions and handle relative edition entry points
_require (function) the require method of the calling module, used to ensure require paths remain correct
customEntry ([string]) an optional override for the entry of an edition, requires the edition to specify a directory property
Returns
void:
